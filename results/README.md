# Reproduced V2 results

This directory is reserved for outputs generated by the current V2 pipeline. No V2 numerical result is currently claimed in the repository.

Before publishing results:

1. Run the complete MATLAB unit-test suite.
2. Confirm that train, validation, and test files are independent at speaker, patient, session, and source level.
3. Execute the unchanged documented configuration.
4. Retain every held-out SNR condition and both methods.
5. Review denoising_metrics.csv for finite, aligned results.
6. Inspect waveform and spectrogram QC figures.
7. Listen to shareable audio examples when ethically and legally permitted.
8. Report failures and unfavorable conditions; do not cherry-pick.
9. Record MATLAB release, toolbox versions, hardware, seed, and configuration.
10. Keep private, licensed, or identifiable source recordings out of GitHub.

A result table should state the number and independence of recordings. With the current six-file design, outcomes are engineering evidence only and must not be described as clinical validation or generalizable performance.
